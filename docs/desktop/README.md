# VisionNet Studio - Документация

## Обзор

Десктопное приложение для семантической сегментации изображений с использованием PyTorch и Tkinter GUI.

## Возможности

- **Управление датасетами** - загрузка и анализ наборов данных в формате COCO
- **Обучение моделей** - тренировка нейросетей для семантической сегментации
- **Инференс** - распознавание и сегментация изображений с отображением результатов
- **Логирование** - отслеживание процесса обучения и распознавания

## Требования

- Python 3.12+
- PyTorch 2.10+
- Torchvision 0.25+
- Scipy 1.17+
- OpenCV 4.13+
- NumPy 2.4+
- Pillow 12.1+

## Использование

### 1. Загрузка датасета

1. Перейдите на вкладку "Датасет"
2. Нажмите "Выберите папку набора данных"
3. Выберите папку с файлом `annotations.json` и папкой `images/`

### 2. Обучение модели

1. Перейдите на вкладку "Обучение"
2. Настройте параметры:
   - Эпохи
   - Размер батча
   - Скорость обучения
   - Размер изображения
   - Устройство (CPU/CUDA)
3. Нажмите "Начать обучение"
4. После завершения сохраните модель

### 3. Инференс

1. Перейдите на вкладку "Распознование"
2. Загрузите обученную модель (`.pth` файл и `model_config.json`)
3. Выберите изображение для распознавания
4. Просмотрите результаты: исходное изображение, маску сегментации и наложение

## Модель

Приложение использует **DeepLabV3** с ResNet50 в качестве backbone:

- Предобученные веса на ImageNet
- Поддержка множественных классов сегментации
- Mixed precision training (FP16) для GPU

### Архитектура

```
VisionNetAdapter
├── Backbone: DeepLabV3 + ResNet50
├── Classifier: Заменяемый слой для N классов
└── Auxiliary Classifier: Для промежуточных предсказаний
```

## Формат данных

Датасет должен быть в формате COCO Segmentation:

```
dataset/
├── annotations.json
└── images/
    ├── image1.jpg
    ├── image2.jpg
    └── ...
```

## Конфигурация модели

```json
{
    "model_name": "VisionNetAdapter",
    "in_channels": 3,
    "num_classes": 9,
    "image_size": 512,
    "classes": ["stem", "flower", "fruit", ...]
}
```
